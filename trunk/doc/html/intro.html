XWord

XWord is an interactive crossword solving program with support for all major puzzle formats.

Crossword features

  • Supported formats:
    • puz (Across Lite format)
    • xpf
    • jpz (Crossword Solver format)
    • ipuz (partial)
    • Across Lite text format
  • Diagramless puzzles
  • Rebus puzzles
  • Circled and shared squares
  • Unconventional numbering schemes
  • Clues in directions (and shapes) other than straight Across and Down
  • Formatted clue text
  • Background images
  • Puzzles with scrambled solutions
  • Notes

Program features

  • A timer
  • Checking and revealing letters
  • Printing
  • Autosaving
  • Many configuration options, including
    • Colors in the grid and clue list
    • Fonts
    • Grid navigation / solving behavior
    • Window layout
  • Extensible via the lua programming language. Plugins include the following and more:
    • Puzzle downloader
    • OneAcross and Wikipedia search
    • Enhanced grid navigation, e.g.
      • Find clues that reference the current clue
      • Find the next blank square
