Build info
==========
  Contributed by Cosmin Truta.

  Compiler:
    gcc-4.5.0-1-mingw32
  Library:
    mingwrt-3.17, w32api-3.14
  Build commands:
    gcc -c -DASMV contrib/asm686/match.S
    gcc -c -DASMINF -I. -O3 contrib/inflate86/inffas86.c
    make -f win32/Makefile.gcc LOC="-DASMV -DASMINF" OBJA="inffas86.o match.o"
